# Changelog

All notable changes to TermDash will be documented in this file.

The format is based on Keep a Changelog, and this project uses semantic versioning.

## [0.1.0] - 2026-06-01

### Added

- Packaged TermDash as an installable Python CLI.
- Added `termdash` command entry point.
- Added project metadata in `pyproject.toml`.
- Added README with current project status, installation instructions, configuration docs, and roadmap.
- Added MIT license.
- Added contributing and security documentation.
- Added GitHub Actions CI.
- Added pytest test suite.
- Added Ruff linting configuration.
- Added local configuration support through `~/.termdash.json`.
- Added GitHub issue templates and pull request template.
